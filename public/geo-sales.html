⚡ Quick Start

One API call. Instant results.

Spend more time building, less time wiring. Here's how easy it is to geocode an address with GeoHive.

Forward Geocoding

Pass an address, get back precise lat/lng, formatted address, components, and confidence score. Supports partial matches, autocomplete, and structured queries.

// Response
{
  "lat": 40.7128,
  "lng": -74.0060,
  "formatted": "350 5th Ave, New York, NY 10118",
  "confidence": 0.98,
  "components": {
    "street": "350 5th Ave",
    "city": "New York",
    "postcode": "10118"
  }
}
// Request
curl https://api.geohive.dev/v1/geocode \
  -H "Authorization: Bearer $KEY" \
  -d '{ "q": "350 5th Ave, New York" }'
// JavaScript SDK
import { GeoHive } from 'geohive';

const geo = new GeoHive('$KEY');
const result = await geo.geocode({
  q: "350 5th Ave, New York"
});
console.log(result.lat, result.lng);
